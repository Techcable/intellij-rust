The plugin adds support for Rust language. Enjoy the full IDE experience featuring:
  • Smart coding assistance
  • Seamless Cargo support
  • Easy access to remote environments
  • Code coverage tooling
  • A fully-fledged Rust debugger (in CLion, IntelliJ IDEA Ultimate, PyCharm Professional, GoLand, and Rider)
  • CPU profiling (in CLion only)
  • Support for Valgrind Memcheck (in CLion only)
